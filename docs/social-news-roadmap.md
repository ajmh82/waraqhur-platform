# Social News Roadmap

## Current Direction

Waraqhur is evolving from a structured publishing platform into an Arabic social news platform.

It will preserve:
- categories
- sources
- editorial clarity
- admin control

It will add:
- timeline logic
- follow relationships
- replies
- reposts
- likes
- bookmarks
- notifications
- search and trending
- external source ingestion such as Nitter

## Product Formula

Waraqhur = Arabic news identity + Twitter-like interaction model + source/category structure

## Near-Term Product Goal

The next implementation steps should move the current product toward:

- a stronger news homepage
- a stronger timeline
- source-first and category-first browsing
- user-level interaction primitives
- future mobile app compatibility
- better public product identity

## Architecture Rule

All future social features should be added on top of the current architecture without breaking:
- auth
- roles
- invites
- notifications
- admin
- audit log
- mobile-readiness structure

## Working Rule

No destructive rebuilds.
Only additive, layered, professional evolution.
